# Hand Gesture Laptop Control System

This project allows users to control their laptop using hand gestures through a webcam.

## Features
- Real-time hand tracking
- Cursor control using index finger
- Pinch gesture for mouse click
- Gesture-based pause and resume
- Touchless computer interaction

## Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## How It Works
1. Webcam captures live video
2. MediaPipe detects hand landmarks
3. Finger positions determine gestures
4. Gestures control mouse and system actions

## Run the Project

Install required libraries:

pip install opencv-python mediapipe pyautogui numpy

Run the program:

python mouse_control.py

## Applications
- Touchless laptop control
- Human Computer Interaction
- Accessibility technology
